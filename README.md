# ⭐ 앙스타 유닛별 최애캐 고르기 | Enstars Unit Bias Chart

👉 **[지금 해보기](https://chillingdaisydev.github.io/enstars-unit-bias-chart/)**

## 기능

- 🎯 **유닛별 최애캐 선택** — 16개 유닛 + 기타 캐릭터
- 🖼️ **이미지 저장** — 선택 결과를 PNG로 저장
- 🔗 **링크 공유** — URL로 선택 결과 공유
- 🌏 **다국어 지원** — 한국어 / 日本語 / English (브라우저 설정 자동 감지)
- 📱 **PWA** — 홈 화면에 앱으로 추가 가능
